
# Installing VirtualBox on Host Machine

## Objective
The objective of this task was to install a virtualization platform to create and manage multiple virtual machines for cybersecurity lab practice.

---

## Software Used
- Oracle VM VirtualBox
- Windows 10 Host Machine

---

## Installation Steps

### Step 1: Download VirtualBox
Downloaded the latest version of VirtualBox from the official website:

https://www.virtualbox.org/

---

### Step 2: Run Installer
Executed the VirtualBox setup file and followed the installation wizard.

Selected default installation options:
- VirtualBox Application
- USB Support
- Networking Features
- Python Support

---

### Step 3: Network Configuration
Allowed VirtualBox to install virtual network adapters required for communication between virtual machines.

---

### Step 4: Complete Installation
Finished installation and restarted the system if required.

---

## Verification
Successfully launched VirtualBox and verified:
- VirtualBox Manager opened correctly
- Virtual machine creation option available
- Network adapters installed successfully

---

## Outcome
VirtualBox was successfully installed on the host machine and prepared for creating Kali Linux, Windows 10, and Metasploitable2 virtual machines.

---

## Skills Learned
- Virtualization basics
- Virtual machine management
- Host and guest environment setup
- Virtual networking fundamentals
